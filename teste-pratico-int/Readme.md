## Questão 2

Escreva um algoritmo que recebe um número **n** e uma base **b**.

Seu algoritmo deve imprimir o logaritmo de **n** na base **b**.

* **Entrada:**

    O programa recebe **n** e **b**

* **Saída:**

    A saı́da consiste de imprimir o logaritmo de **n** na base **b**.

* **Exemplo de Entrada:**

    4096 2

* **Exemplo de Saída:**

    12
